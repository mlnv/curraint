Features · everything that keeps you in flow

Nothing breaks
the current.

01

Streaming, with a stop button

Responses show up as they're generated. If the model starts rambling or you've seen enough, hit stop and keep what you've got.

02

Edit any message, regenerate

Realized you asked the wrong thing five messages ago? Edit that message and the thread picks up from there. No starting over.

03

Your choice of provider

OpenAI, DeepSeek, GitHub Copilot, or run LM Studio locally. Anything with an OpenAI-compatible endpoint works. Switch whenever you want.

04

Context safety built in

Long conversations don't suddenly error out. When the context window fills up, older messages get summarized instead of dropped.

05

Encrypted secrets on disk

Your API keys are encrypted with AES-256-GCM. They never touch disk in plain text. Not in settings, not in logs, nowhere.

06

Save sessions when it matters

Conversations aren't saved unless you turn it on. When you do, you can browse, search, and jump back into old sessions from any interface.

Get started · find your current

Jump in.
The water's fine.

1

You'll need Node.js and pnpm

Make sure you have Node.js 22+ and pnpm 10+ installed. Then grab the code:

git clone https://github.com/curraint/curraint.git
cd curraint
pnpm install
2

Build

Everything builds in one shot:

pnpm build
3

Launch an interface

Desktop
pnpm desktop
CLI
pnpm cli
Obsidian
pnpm --filter @curraint/obsidian-plugin build

Then copy dist/ into your vault's .obsidian/plugins/curraint/

4

Connect a provider

Set an environment variable, or use the settings UI in the desktop app. Works with OpenAI, DeepSeek, LM Studio, GitHub Copilot, and any OpenAI-compatible API.

# Add to your shell profile or .env
CURRAINT_PROVIDER=openai
CURRAINT_API_KEY=sk-...
CURRAINT_MODEL=gpt-4o-mini
